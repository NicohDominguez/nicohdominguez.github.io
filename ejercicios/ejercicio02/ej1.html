Inicio

Bienvenido a mi página web personal. Este sitio fue creado como parte de un trabajo práctico de la materia de desarrollo web.

El objetivo de esta página es mostrar una pequeña presentación sobre mí, mis intereses y los proyectos que estoy realizando actualmente.

También sirve como práctica para aprender a utilizar etiquetas HTML y estructurar correctamente una página web.

Sobre mí

Mi nombre es Nicolás, tengo 18 años y actualmente estoy estudiando una licenciatura en gestión de la tecnología.

Me interesa mucho el mundo de la informática, especialmente el desarrollo web, el diseño de aplicaciones y las nuevas tecnologías.

En mi tiempo libre me gusta aprender sobre programación, investigar herramientas digitales y mejorar mis habilidades en la computadora.

Entre mis habilidades puedo destacar el manejo básico de HTML, el uso de herramientas como Visual Studio Code y el interés constante por seguir aprendiendo.

Uno de mis objetivos es poder desarrollar proyectos más avanzados en el futuro y seguir creciendo en el área tecnológica.

Proyectos

Actualmente estoy trabajando en la creación de mi primera página web utilizando HTML5.

Este proyecto incluye diferentes ejercicios como la creación de textos, listas, imágenes y enlaces.

También estoy aprendiendo a estructurar correctamente una página utilizando etiquetas como header, nav, main, section y footer.

A futuro, me gustaría incorporar estilos con CSS y agregar interactividad con JavaScript para hacer páginas más completas.

Este es solo el comienzo de mi aprendizaje en el desarrollo web.
